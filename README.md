#  Full-Stack Portfolio

Bu proje, bir yazılım geliştiricisi olarak yeteneklerimi, projelerimi ve iletişim bilgilerimi sergilediğim modern, responsive ve performans odaklı bir portfolyo web sitesidir.

##  Kullanılan Teknolojiler

* **Frontend:** React.js
* **Styling:** Tailwind CSS (Modern & Utility-first)
* **Animasyonlar:** CSS Transitions & Hover Effects
* **İkonlar:** Google Material Symbols
* **İletişim Formu:** Web3Forms API 
* **Deployment:** Vercel 

##  Öne Çıkan Özellikler

* **Tam Responsive Tasarım:** 
* **Modern UI/UX:** Karanlık mod (Dark Mode) öncelikli, temiz ve okunabilir arayüz.
* **Dinamik Mobil Menü:** JavaScript state yönetimi ile çalışan akıcı navigasyon.
* **Çalışan İletişim Formu:** API entegrasyonu sayesinde doğrudan e-posta kutuma düşen mesajlar.
